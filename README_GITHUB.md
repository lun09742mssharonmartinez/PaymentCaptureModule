# 支付链接抓取模块

[![Android CI](https://github.com/YOUR_USERNAME/PaymentCaptureModule/actions/workflows/android-build.yml/badge.svg)](https://github.com/YOUR_USERNAME/PaymentCaptureModule/actions)

一个基于 LSPosed/Xposed 框架的支付链接拦截模块，类似 X2.apk。

## ✨ 功能特性

- ✅ Hook 支付宝应用，拦截支付链接
- ✅ 拦截支付宝 SDK 调用（淘宝/美团/京东等支付）
- ✅ 提取支付二维码数据
- ✅ 显示支付金额、订单信息
- ✅ 支持多种支付场景
- ✅ 模块管理界面

## 📥 下载 APK

### 自动编译版本
访问 [GitHub Actions](https://github.com/YOUR_USERNAME/PaymentCaptureModule/actions) 页面下载最新编译的 APK

### 手动编译
参见 [BUILD.md](BUILD.md)

## 🔧 环境要求

- Android 8.0+ (API 26+)
- 已 Root 的设备
- LSPosed 框架 或 Xposed 框架
- Magisk (推荐)
